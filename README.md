1. Look at the two example scripts (`example_http_check.py` and `example_port_check.py`)
2. Create your own branch: `git checkout -b your_vuln_name`
3. Write a Python script that finds and demonstrates a vulnerability
4. Submit your finding at https://submit.0x10.cloud
5. Merge your branch into master when done
